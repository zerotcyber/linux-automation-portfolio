# Linux Super-Automation Suite

## Description
This project contains bash scripts to automate daily linux administration tasks:
1. **User Onboarding**: Bulk creation of users from a list.
2. **System Health**: Monitors disk and RAM usage.
3. **Log Maintenance**: Archives and compresses old logs.
4. **Backup**:Timestamped directory backups.

##Usage
Run the scripts from the 'scripts/' directory with sudo permissions.
